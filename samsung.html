Switcher Timer

Overview

스위처의 주요 지표인 유료 구독 전환율을 높히기 위해 진행한 두 번의 스프린트.

Duration

August 2016-September 2016, Feb 2017 - Mar 2017

스위처는 부착형 스마트 컨트롤 하드웨어 제품입니다. 모바일 어플리케이션과 블루투스로 연결한 후 조명을 끄고 켜거나, 설정할 수 있는 기능을 제공했습니다.

스위처의 주 고객은 20대 초반에서 30대 중반이며 1,2인 가구의 거주 형태를 띄고 있었습니다.

Then, What is Switcher Timer?

블루투스를 통해 모바일 앱과 스위처가 연결 되어있을 때, 유저는 기기 안에 타이머를 생성하고 저장할 수 있습니다. 이렇게 저장하고 난 뒤, 블루투스 연결을 끊고 나면 정해진 시간에 타이머가 자동으로 작동하게 됩니다.
유료 구독 전환율에 큰 영향을 미치는 요인을 찾기 위해 서베이, GA 분석, 인터뷰 등 정성과 정량 조사를 진행하였고 아래와 같은 두 가지 인사이트를 찾을 수 있었습니다.

  1. 반납한 유저 중 35.3% 만이 스위처 타이머를 사용한 것에 비해, 유료 구독으로 전환한 유저의 67.5% 가량이 스위처 타이머를 사용하고 있었습니다.
  2. 유료 구독 전환 고객 중, 타이머를 사용하고 있는 유저들은 더 비싸고 긴 장기플랜을 선택하고 사용하고 있었습니다.

Hypothesis 1

더 많은 유저가 타이머를 사용한다면 유료 구독 전환율이 올라 갈 것이다.

Based on our hypothesis, the product team defined the higher level goals and released two sequential features about Timer.

Higher Level Goals

📈 Business Goals

To increase the revenue: more Triennial Recurring Revenue as well as the average price of first payment after a free-trial ↗

😣 User Goals

To understand the value Switcher can provide in their daily chores.

  2. Not enough trust in the new connected device. They were worried if Timer didn't work while they cannot be physically next to the device.

Sprint 2

Interactive Onboarding Guide

Redefine User Problem

  1. 94.9% of users set a timer as soon as they paired smartphones and Switcher devices for the first time.
  2. Correlation between the number of timers user set and the subscription plan type.

User Problems

  1. User can think of only one usecase: Turn lights as a wake up alarm
  2. User can think about only one usecase: Turn lights as a wake up alarm

Solution

  1. Suggest multiple key usecases to help users to setup more than two timers.
  2. Help users to set multiple timers at once
