Building a Source of Truth

I synthesized core features from several previous user interviews and collected user requests related to collaboration. I also reviewed the holistic flows from prototyping in ProtoPie Studio to interacting with prototypes in ProtoPie Player(Mobile Application Viewer). These activities helped me to drew the first rough information architecture and utilize it to talk about cloud architecture.

I kept updating it repeatedly as we made progress in scope and the scope of the project. It performed as a source of truth for all stakeholders until we created user stories for development.
